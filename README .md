
# 🧾 Insurance Data Visualization Project

## 📘 Overview
This project explores and visualizes **Insurance Data** using Python. It helps to uncover insights such as the relationships between **age, gender, BMI, region, smoking habits**, and **insurance charges**. The analysis is done using **Pandas**, **NumPy**, and **Matplotlib/Seaborn** for data visualization.

---

## 📂 Dataset
The dataset used contains the following key columns:

- `age` — Age of the policyholder
- `sex` — Gender of the policyholder
- `bmi` — Body Mass Index (BMI)
- `children` — Number of dependents
- `smoker` — Whether the policyholder is a smoker or not
- `region` — Residential area in the US (northeast, southeast, southwest, northwest)
- `charges` — Medical insurance cost

---

## ⚙️ Technologies Used
- **Python 3.13.0
- **Pandas** – Data cleaning and manipulation
- **NumPy** – Numerical operations
- **Matplotlib & Seaborn** – Data visualization and plotting

---

## 📊 Key Visualizations
1. Distribution of **insurance charges**
2. Relationship between **BMI** and **charges**
3. Comparison of **charges** by **smoking status**
4. Average **charges by region**
5. Correlation heatmap among all numerical variables

---

## 🧠 Insights
- Smokers tend to have significantly higher insurance charges than non-smokers.
- BMI and age have a strong positive correlation with insurance charges.
- Regional differences in insurance costs are relatively minor compared to lifestyle factors.
